# 冒泡排序
> 它重复地走访过要排序的数列，一次比较两个元素，如果他们的顺序错误就把他们交换过来。走访数列的工作是重复地进行直到没有再需要交换，也就是说该数列已经排序完成。

时间复杂度：  
最好 O(n)   （在设置了无交换标志位的前提出现）  
最坏 O(n^2)  
平均 O(n^2)  

```
class BubbleSort:
    def bubbleSort(self, A, n):
        # write code here
        for i in range(0, n):
            for k in range(0, n-1):
                if A[k] > A[k+1]:
                    A[k], A[k+1] = A[k+1], A[k]
        return A  
```
